# Contributing to Vidstream

First off, thank you for considering contributing to Vidstream! It's people like you that make it a great tool.

## How Can I Contribute?

### Reporting Bugs
* Check the existing issues to see if it has already been reported.
* If not, open a new issue with a clear title and description.
* Provide as much information as possible, including steps to reproduce.

### Suggesting Enhancements
* Open a new issue and describe the enhancement you'd like to see.
* Explain why this enhancement would be useful.

### Pull Requests
* Fork the repository.
* Create a new branch for your feature or bug fix.
* Make your changes and commit them with descriptive messages.
* Push your changes to your fork.
* Open a pull request against the `master` branch.

## Style Guidelines
* Follow the official [Flutter style guide](https://dart.dev/guides/language/effective-dart/style).
* Use descriptive variable and function names.
* Comment your code where necessary.

Thank you for your help!
